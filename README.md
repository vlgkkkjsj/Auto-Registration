# Automating Product Registration

This Python project automates product registration based on data stored in a database or spreadsheets.

## Features

- **Automatic Registration:** Captures data from the database table and registers products automatically.
- **Continuous Update:** Maintains synchronization between the database and the application for real-time updates.
- **Flexible Configuration:** Customize the settings to adapt to different table structures.

## Requirements

- Python 3.x
- Required Python libraries (listed in the requirements.txt file)
